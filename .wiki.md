# Project Summary
The project is developing an elegant banking website called **Bank Nusantara**. This website aims to provide a user-friendly online banking experience with a focus on modern financial solutions, featuring a green color theme for branding. The site includes multiple pages such as a homepage, product listings for 36 banking products, a vision and mission page, and social media links in the footer, all implemented using HTML, CSS, and JavaScript without frameworks.

# Project Module Description
- **Homepage (index.html)**: The main landing page that introduces the bank and its services.
- **Product Page (produk.html)**: Displays 36 different banking products including savings accounts, loans, investments, and insurance.
- **Vision and Mission Page (visi-misi.html)**: Outlines the bank's commitment and core values.
- **CSS (style.css)**: Styles for the website, ensuring a responsive and modern design.
- **JavaScript (script.js)**: Contains functionality for navigation, animations, product filtering, and smooth scrolling.

# File Description Inventory
- **index.html**: The main entry point of the website featuring the bank's overview and services.
- **produk.html**: Lists various banking products with descriptions and features.
- **visi-misi.html**: Describes the bank's vision, mission, and values.
- **style.css**: Contains all CSS styles for layout, colors, and responsive design.
- **script.js**: Handles interactive features and dynamic content on the website.
- **template_config.json**: Configuration settings for templates used in the project.
- **todo.md**: A markdown file for keeping track of tasks and project structure.

# Technology Stack
- **HTML5**: For structuring web pages.
- **CSS3**: For styling and layout, using Flexbox and Grid.
- **JavaScript**: For client-side scripting and interactivity.

# Usage
1. Install dependencies using package manager.
2. Build the project using the build command.
3. Run the project to view the website in a browser.
