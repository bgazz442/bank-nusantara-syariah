# Website Bank Elegan - Todo List

## Struktur File yang Dibutuhkan:
1. **index.html** - Halaman utama dengan hero section, layanan utama, dan footer dengan social media
2. **produk.html** - Halaman produk dengan 36 produk perbankan
3. **visi-misi.html** - Halaman visi dan misi bank
4. **style.css** - Styling dengan tema hijau yang elegan dan responsive
5. **script.js** - JavaScript untuk interaktivitas dan navigasi

## Fitur Utama:
- Design elegan dan profesional untuk bank
- Warna hijau sebagai tema utama (judul dan aksen)
- Navigasi yang jelas antar halaman
- Responsive design
- Footer dengan social media links dan kredit
- 36 produk bank yang beragam (tabungan, kredit, investasi, dll)

## Implementasi:
- Menggunakan HTML5 semantic elements
- CSS Grid dan Flexbox untuk layout
- JavaScript untuk smooth scrolling dan interaktivitas
- Mobile-first responsive design
